## Project Objectives
### Problem
Students need a way to communicate and share course resources with each other outside the class effictively, but using traditional social media platforms might be affected scattered resources, messages from unwanted and supsipusious ads.
### Objectives
We aspire to acheive these objectives bu the end of this project:
- add the ability for students to make course groups to reduce their time to find course group messages(TFCGM) after 3 months.
- add the ability to rate resources to increase the quality of the resources shown in the begining of the course resources section after 3 months.
- add the ability to detect and remove spam messages to reduce the number of spam messages in course groups(NSM).
