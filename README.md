User Story 1
Description
As a student I want to browse all university groups so that I can find my university's group.

Acceptance Criteria
Given that I am logged in, When I access the platform, Then I should see a list of universities with their groups.
Given that I select a university, When I click on its group, Then I should see the list of college groups inside it.

User Story 2
Description
As a student I want to upload or access resources inside the course group so that I can study and share materials.

Acceptance Criteria
Given that I am inside a course group
When I open the resources section
Then I should be able to upload and download resources.

User Story 3
Description
As an admin I want to create and manage university, college, and course groups so that the platform remains organized.

Acceptance Criteria
Given that I am an admin
When I create or edit a group
Then the changes should appear immediately to students

Description
As a student I want to receive notifications about new resources or messages in my groups so that I stay updated.

Acceptance Criteria
Given that a new resource or message is added
When I am a member of the group
Then I should receive a notification.

Description
As a student I want to view all members of my course group so that I know who is in the group.

Acceptance Criteria
Given that I am inside the group
When I open the members list
Then I should see all students in the group.
